This project is a front-end & UX prototype for an emotional support product.

The focus is on interaction design, information structure, and gentle user flows.

Only key scenarios are fully implemented. Other sections are intentionally left as prototypes.









🏛️ 猗窩座的房間：文字互動式自我修復空間
這是一個專為「情緒收納」設計的數位房間。當使用者感到痛苦或挫敗時，這裡提供一個安靜、非評判的空間，讓負面情緒能被命名並被溫柔地接住 。


🌟 核心功能

情緒命名與釋放：使用者可以輸入當下最刺痛的一句話（例如：「我現在最痛的是...」），將抽象的痛苦轉化為具體的文字 。



非評判式角色回覆：系統會以「理解與接住」的口吻給予即時回饋（例如：「我感受到了，那份沉重並不容易」），提供情緒上的支持與共感 。



「回家牆」情緒存檔：利用瀏覽器儲存技術，將使用者的輸入記錄在牆上。這不僅是紀錄，更是一種「我已經走過來了」的視覺化證明 。


🎨 UI/UX 設計特色

沉浸式視覺：採用深色調與夜燈感的設計，降低視覺壓力，營造願意停留的「房間感」 。


溫暖的互動節奏：特別設計了打字機效果，模擬真人對話的節奏感，讓文字出現的過程顯得更有溫度與誠意 。



極簡低負擔：刻意減少複雜的操作，讓處於情緒低潮的使用者能以最低的認知負擔獲得安慰 。

🛠️ 技術實作

JavaScript 動態操作：實現即時的文字輸入處理與隨機語句抽選邏輯 。


LocalStorage 持久化：即便重新整理網頁，使用者的歷史情緒紀錄依然會被保存在「回家牆」上 。



CSS 動畫：運用淡入（Fade-in）效果提升沉浸式體驗 。
